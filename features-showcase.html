üìù Available Shortcodes

Embed tracker features anywhere on your website

[tracker_list]

Description: Display list of all torrents with search and pagination

Parameters:

  • count="30" - Torrents per page
  • orderby="created_at" - Sort field
  • order="DESC" - Sort direction
  • search="1" - Show search form
  • pagination="1" - Show pagination
  • columns="name,size,seeders,leechers" - Display columns

[tracker_stats]

Description: Display tracker statistics and network overview

Parameters:

  • layout="grid|table" - Display format
  • show="torrents,peers,seeders,leechers" - Stats to show

[tracker_user_stats]

Description: Display current user's tracker statistics

Parameters:

  • user_id="current" - User ID (default current)
  • show_passkey="1" - Display passkey
  • show_active="1" - Show active torrents

[tracker_latest]

Description: Display newest torrents

Parameters:

  • count="10" - Number to display
  • show_stats="1" - Show peer stats

[tracker_upload]

Description: Embedded torrent upload form

Parameters:

  • redirect="url" - Redirect after upload
  • category="required" - Require category selection

[tracker_torrent]

Description: Display single torrent information

Parameters:

  • id="123" - Torrent ID
  • hash="abc123" - Info hash
  • show_files="1" - Display file list
  • show_peers="1" - Show peer stats

[tracker_user_torrents]

Description: Display torrents uploaded by specific user

Parameters:

  • user_id="5" - User ID
  • count="10" - Number to display

[tracker_status]

Description: Display tracker online/offline status

Parameters:

  • show_url="1" - Show announce URL
  • refresh="30" - Auto-refresh interval (seconds)

üóÑÔ∏è Database Tables

Plugin uses WordPress database with custom tables for torrent tracking

wp_tracker_torrents

Main torrent information table storing torrent metadata

  • ID, Info Hash, Filename, Name
  • File Size, Seeders, Leechers, Completed
  • Description, Category, Created/Updated timestamps

wp_tracker_peers

Active peer information for real-time tracking

  • Info Hash, User Passkey, Peer ID
  • IP Address, Port, Uploaded/Downloaded bytes
  • Status (seeding/leeching), Last announce time

wp_tracker_files

Individual files within torrents

  • Torrent ID, File path, File size
  • MD5 hash, MD4 hash, SHA1 hash
  • Piece length, Pieces information

wp_tracker_history

Historical tracking data for analytics

  • Announce records, Event type
  • Downloaded/uploaded bytes, Timestamp
  • Peer statistics, User information

üìñ Implementation Guide

How to implement these features in your project

Step 1: Install Plugin

  1. Download WP Torrent Tracker plugin ZIP
  2. Upload to /wp-content/plugins/
  3. Activate from WordPress admin panel
  4. Configure basic settings (announce URL, intervals)

Step 2: Install Theme

  1. Download TorrentBlog theme ZIP
  2. Upload to /wp-content/themes/
  3. Activate from Appearance ‚Üí Themes
  4. Customize via Appearance ‚Üí Customize

Step 3: Create Pages

  1. Create "Browse" page with [tracker_list] shortcode
  2. Create "Upload" page with [tracker_upload] shortcode
  3. Create "Profile" page with [tracker_user_stats] shortcode
  4. Create "Dashboard" page with [tracker_stats] shortcode

Step 4: Configure Tracker

  1. Set primary announce URL in settings
  2. Configure announce interval (default 1800s)
  3. Enable/disable features (scrape, private flag check)
  4. Customize appearance settings

Step 5: Test & Deploy

  1. Test announces with torrent client
  2. Verify peer tracking in admin panel
  3. Test user registration and passkey generation
  4. Deploy to production server
